Vocabulary

Words
Defintition
Web Form
Allows the creation of interactive websites for user feedback
Controls/Widgets
Objects that allow a user to interact with A form
Data Field
Stoes the data values supplied by a user
Input boxes
Boxes used to insert text and numeric values
Option/raido buttons
Buttons to select data values from a predefined set of options
Selection Lists
List that allows the user to select data values from an extensive list of options
Check boxes
Allows the user to select data values limited to 2 possibilities, such as "yes" or "no"
Text area boxes
Input boxes used to enter multiple text strings that may include several lines of content
Spin boxes
Widget that allows users to enter integer values confined to a specified range
Slider boxes
Widget that allows users to enter numeric values confined to a specified range
Calendar controls
Widget that allows users to select date and time values
Color Pickers
Widget that allows users to choose color values
Get Method
Tells the browser to append the form data to the end of the URL specified in the action attribute (Default Method)
Post Method
Sends the form data in its own separate data stream (Considered to be a more secure form of data transfer)
