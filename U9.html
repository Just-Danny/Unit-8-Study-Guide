Unit 9 Study Guide


Vocabulary

Words
Defintition
Web Form
Allows the creation of interactive websites for user feedback
Controls/Widgets
Objects that allow a user to interact with A form
Data Field
Stoes the data values supplied by a user
Input boxes
Boxes used to insert text and numeric values
Option/raido buttons
Buttons to select data values from a predefined set of options
Selection Lists
List that allows the user to select data values from an extensive list of options
Check boxes
Allows the user to select data values limited to 2 possibilities, such as "yes" or "no"
Text area boxes
Input boxes used to enter multiple text strings that may include several lines of content
Spin boxes
Widget that allows users to enter integer values confined to a specified range
Slider boxes
Widget that allows users to enter numeric values confined to a specified range
Calendar controls
Widget that allows users to select date and time values
Color Pickers
Widget that allows users to choose color values
Get Method
Tells the browser to append the form data to the end of the URL specified in the action attribute (Default Method)
Post Method
Sends the form data in its own separate data stream (Considered to be a more secure form of data transfer)
Field set
Gruop fiels that share a common purpose
Legend
Describes the content of a field set (Like A Caption)
Virtual Keyboards
Software Representations of a physical device
Spinner Control
Displays an up or down arrow to increase/decrese the field value by a set amount
Range Slider
Limits a numeric field to a range of possible values
Data List
A list of possible data values that form a field can have
Form Button
A type of form control that performs an action
Command Button
Runs a program that affects the content of a page or the actions of a browser
Submit Button
Submits a form to the server for processing when clicked
Reset Button
Resets a form, changing all fields to their values and deleting any field values that a user has entered
Validation
Process of ensuring that a user has supplied valid data
Server-side Validation
Validation occurs on the web server (After you Submit)
Client-side Validation
Validation occurs in the user's browser (Before you submit)
Regular expression (Regex)
A concise description of a character pattern
Inline Vlidation
The technique of immediate data validation and reporting of errors
Focus
The state in which an element has been clicked by the user, making it the active control on the form

Attributes
Attribute Defintition
Action Provides the location of the web server program that processes the form
Method Specifies how the browser should send form data to the server
Enctype Specifies how the form data should be encoded as it is sent to the server
name Provides the name of the data field associated with the control
ID Identifies the control in which the user enters the field value
Type Indicates the data type of the field
value Specifies a default field value
Placeholder Text strings that appear within a form control, providing a hint about the kind of data that should be entered into a field
Size Displays a selection list as a scroll box
Multiple Allows more than one item to be selected in a selection list
Rows Specifies the number of lines in the text area box
Cols Specifies the number of characters per line
Required Identifies what data fields are required
Pattern Validates a text value against a regular expression
Maxlength Sets the maxinum number of characters in a field value (Doesn't distinguish between characters and digits
